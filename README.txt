# Currency-Calculator

Aby otworzyć program
1. Proszę zaimportować projekt do dowolnego środowiska(IDE) np. IntelliJ IDEA lub uruchomiź plik xml
2. Otworzyć klasę main oraz uruchomić.
3. Program zawiera instrukcję co robić dalej, aby uzyskać wynik.
4. Komendy nalezy wpisywac do konsoli oraz zatwierdzać przyciskiem ENTER.
